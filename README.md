# Projeto - Formulário de Cadastro

Projeto simples de formulário de cadastro em HTML, CSS e JavaScript.

## Estrutura do projeto

- `index.html` - página principal do formulário
- `assets/css/style.css` - estilos do formulário
- `assets/img/img-container.svg` - imagem usada na página

## Como usar

1. Abra `index.html` no navegador.
2. O formulário funciona localmente com validação básica e modal de confirmação.

## Observações para GitHub

- O projeto foi reorganizado para ter `index.html` na raiz do repositório.
- A pasta `assets/` contém os estilos e imagens usados pela página.
- Este layout facilita o deploy no GitHub Pages usando a raiz do repositório como fonte.
